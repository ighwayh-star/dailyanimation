# 🌸 每日动画放送

基于 [Bangumi 番组计划](https://bgm.tv) 数据的浏览器扩展插件，优雅展示每日更新的动画番剧。

---

## ✨ 功能

- **每日放送** — 自动展示今天 & 昨天两天更新的番剧
- **详情直达** — 点击卡片在新标签页打开 Bangumi 条目页
- **自由隐藏** — 不追的番剧一键隐藏，眼不见为净
- **毛玻璃 UI** — 樱花粉渐变、飘落花瓣、玻璃拟态卡片
- **智能缓存** — 1 小时内不重复请求，后台每 30 分钟自动刷新
- **数据来源** — Bangumi 公开 API，无需登录

---

## 📦 安装

### Chrome / Edge

1. 打开扩展管理页面：
   - **Chrome** → `chrome://extensions`
   - **Edge** → `edge://extensions`
2. 打开右上角 **「开发者模式」** 开关
3. 点击 **「加载已解压的扩展程序」**
4. 选择 `dailyanimation` 文件夹
5. 完成 ✅

### Firefox（需要手动适配）

Firefox 支持 Manifest V3，但需要额外步骤：

1. 将 `manifest.json` 中 `"service_worker"` 改为 `"scripts": ["background/service-worker.js"]`
2. `about:debugging` → 「此 Firefox」→ 「临时载入附加组件」→ 选择 `manifest.json`

## 🎮 使用

| 操作 | 方式 |
|------|------|
| 打开插件 | 点击浏览器工具栏的插件图标 |
| 查看番剧详情 | 点击番剧卡片（新标签页打开 Bangumi） |
| 隐藏番剧 | 点击卡片右侧 👁 按钮 |
| 取消隐藏 | 展开底部「已隐藏」分区，点击 👁‍🗨 按钮 |
| 手动刷新 | 点击右上角 🔄 按钮 |

## ❓ 常见问题

**Q: 点击图标没反应 / 弹窗空白？**
A: 右键插件图标 → 「检查弹出窗口」查看控制台是否有错误。通常是网络问题导致 API 请求失败。

**Q: 数据显示不全 / 某天没番剧？**
A: Bangumi 日历只返回当前季度播出的番剧。如果某天确实没有新番播出，会显示「暂无更新」的空状态。

**Q: 星期显示不对？**
A: 使用的是你电脑的本地时间。确认系统日期是否正确。

**Q: 如何彻底删除已隐藏的记录？**
A: 隐藏列表存储在浏览器本地。清除浏览器数据或卸载重装会重置，也可以右键插件 → 「检查弹出窗口」→ 控制台执行：
```js
chrome.storage.local.clear()
```
